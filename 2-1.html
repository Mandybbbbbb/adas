糖醋排骨

材料：

<
    <
  • 豬小排（500克）
  • 生薑（一小塊，切片）
  • 大蒜（2瓣，切碎）
    • <
  • 紅椒（一個，切絲，可選）
  • 青椒（一個，切絲，可選）
    • <
  • 糖（3湯匙）
    • <
  • 醋（2湯匙）
  • 生抽（2湯匙）
    • <
  • 料酒（1湯匙）
  • 番茄醬（2湯匙）
    • <
  • 水淀粉（適量，用於勾芡）
    • <
  • 食用油（適量，用於炸排骨）
    • <
  • 鹽和胡椒（根據個人口味調味）

    • 步驟：

    1. 豬小排洗凈切塊，用鹽和胡椒調味，稍微腌制片刻。
      2. <
    2. 熱油鍋，將小排炸至金黃酥脆，撇去浮沫，瀝乾多餘的油分。
      3. <
    3. 熱鍋冷油，加入姜片和蒜蓉爆炒香。
      4. <
    4. 將糖、醋、生抽、料酒、番茄醬加入鍋中，攪拌均勻。
    5. 加入炸好的豬小排，煮至醬汁收干且均勻裹在排骨上。
      6. <
    6. 加入紅椒絲和青椒絲，炒勻即可。
    7. 如果想要醬汁更濃稠，可以加入水淀粉勾芡。
      8. <
    8. 糖醋排骨完成，上桌享用！

    典故

    糖醋排骨最早的传说最为有趣，并与起源地有关系，或许是为了挣得这道菜的籍贯而编撰的。
    说的是无锡当年有道叫做“肉骨头”的菜，名气很大，最早创立于宋朝。
    那时还不叫糖醋排骨，后来从肉骨头演变而来。
